# RH_PilotSets - تنظیمات Expert Advisor برای طلا (XAUUSD)

> ⚠️ **توجه:** این ریپازیتوری فقط شامل **فایل تنظیمات (`.set`)** یک ربات معاملاتی است و **کد منبع** خود Expert Advisor در آن قرار ندارد. برای استفاده، به فایل‌های اجرایی (`.ex4` یا `.mq4`) مربوطه نیاز دارید.

---

## 📌 معرفی پروژه

این مخزن، فایل پیکربندی مربوط به یک **Expert Advisor (EA)** در پلتفرم متاتریدر ۴ (MT4) است که برای معامله‌گری **جفت‌ارز طلا (XAUUSD)** در **تایم‌فریم ۱ دقیقه** طراحی شده است.

استراتژی به کار رفته، مبتنی بر مفاهیم **Smart Money Concepts (SMC)** و **Price Action** بوده و از ابزارهایی مانند **FVG (Fair Value Gap)**، تشخیص ساختار شکسته‌شده و الگوهای کندلی برای ورود و خروج استفاده می‌کند.

## 🛠 پیش‌نیازها و نصب

### ۱. نرم‌افزار مورد نیاز
- **پلتفرم متاتریدر ۴** (نسخه‌های جدیدتر)
- **Expert Advisor متناظر** (که این فایل تنظیمات برای آن نوشته شده است)

### ۲. نحوهٔ نصب فایل تنظیمات
1. فایل `setNo1_xauusd_m1.set` را دانلود کنید.
2. آن را در پوشه‌ی زیر کپی کنید (بسته به نصب MT4 شما):
   - `MQL4/Profiles/` یا
   - `MQL4/Presets/` (در برخی نسخه‌ها)
3. در متاتریدر، EA را روی نمودار XAUUSD در تایم‌فریم M1 قرار دهید.
4. از منوی Properties (یا راست‌کلیک > Properties) گزینه‌ی **Load** را بزنید و فایل `.set` را انتخاب کنید.
5. تنظیمات خودکار اعمال می‌شوند.

> 📌 در صورتی که EA شما از پسوند `.set` پشتیبانی نمی‌کند، ممکن است نیاز به وارد کردن دستی پارامترها داشته باشید (لیست پارامترها در بخش بعدی آمده است).

## ⚙️ توضیحات پارامترهای کلیدی

| پارامتر | مقدار | توضیح |
|---------|-------|--------|
| `tradeMagic` | 123456 | شماره جادویی جهت شناسایی معاملات این ربات |
| `buyActive` / `sellActive` | true | هر دو سمت خرید و فروش فعال هستند |
| `trigger_maxTrade` | 10 | حداکثر تعداد معاملات همزمان |
| `stopBufferPoint` | 51.0 | حد ضرر اولیه بر اساس پیپ (۵۱ پیپ) |
| `maxStopPip` | 4000 | سقف حد ضرر (به عنوان کران بالا) |
| `tptype` | 1 | نوع استراتژی حد سود (احتمالاً مبتنی بر ساختار) |
| `smt_active` | false | تکنیک SMT (واگرایی بین دو دارایی) غیرفعال است |
| `imbalanceGap_useInFvg` | true | استفاده از مفهوم FVG و عدم تعادل قیمت |
| `trailingAtr_active` و مشابه | false | تمام تریلینگ استاپ‌ها غیرفعال هستند |

🔍 سایر پارامترهای مرتبط با ساختار قیمت (مانند `trigger_totCdl`، `failStructure_closeUnderLeg` و ...) نشان‌دهندهٔ وابستگی استراتژی به شکست ساختارها و الگوهای کندلی است.
